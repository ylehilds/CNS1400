         CNS 1400: Lab Seven

Objectives:
The objective of this lab is to help you understand how pass by value and pass by reference work. 

After completing this lab, you should be able to come up with solutions to problems that use programmer written functions where values can be passed by value and/or by reference. You should be able to write proper function prologues and use the assert macro to test preconditions.
