Objectives:
The objective of this lab is to build your understanding of dynamic allocation and demonstrate the use of a dynamically allocated array. 

After completing this lab, you should be able to 

Explain how dynamic allocation works. 
Tell where in memory dynamically allocated data is stored. 
Write a program that dynamically allocates an array. 
Explain how to release dynamicaly allocated memory.
